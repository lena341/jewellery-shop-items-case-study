    The website is divided in sections. The first and third include information about the house. The first has a light blue background to help stand out from the rest of the page and a link that leads to a new page. The second is an item slider where each item has its own description and link that leads to product page shown with CSS3 overlay on hover. The last includes links for services. The tools used are HTML5, CSS3, Sass and Bootstrap 3. Everything that is displayed on the website can be customized according to your needs and new sections can be added or removed. You can download the code from my GitHub, customize it or contact me for help. Don't forget to credit me.
