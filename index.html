    The purpose is to give information about a specific item. The images of the item are displayed with a carousel, the description and details in tabs. In another section there is a slider with similar items and their description shown on hover again made with CSS3 overlay and in the last section some more information about the house's jewellery. The tools used are HTML5, CSS3, Sass and Bootstrap 3. Everything that is displayed on the website can be modified according to your needs and new sections can be added or removed. You can download the code from my GitHub, customize it or contact me for help. Don't forget to credit me.
